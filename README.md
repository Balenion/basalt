# AuthForge 🛡️

**AuthForge** — специализированный инструмент для разработчиков и специалистов по безопасности, предназначенный для генерации надежных криптографических ресурсов. Программа фокусируется на создании устойчивых к взлому паролей и секретных ключей, необходимых для современных протоколов авторизации.

### Функциональные возможности 

- ​**Key Generation**: Создание высокоэнтропийных секретных ключей для JWT (JSON Web Tokens), API-интерфейсов и других криптографических нужд.  
- ​**Password Engineering**: Хеширование паролей настраиваемой сложности с использованием соли (Salting) и для максимальной защиты от перебора.
- **Batch Processing & Excel Integration**: 
	- **Импорт**: Загрузка списка паролей из Excel-файла. Доступен встроенный шаблон для корректного заполнения.
	- **Обработка**: Массовое хеширование по выбранному алгоритму (Argon2id, BCrypt и др.).
	- **Экспорт**: Выгрузка результатов в формате XLSX с помощью библиотеки ClosedXML для дальнейшего использования в базах данных или системах миграции.
- **​Zero Knowledge**: Приложение работает полностью в автономном режиме. Никакие входные данные, сгенерированные пароли или ключи не сохраняются и не передаются по сети.
